Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a way to represent a webpage so that programming languages like JavaScript can interact with it. It treats the structure of the HTML or XML document as a tree of objects, where each element (like paragraphs, headers, and links) can be accessed, modified, or deleted. This allows developers to dynamically change the content, structure, and style of a webpage after it has loaded.

How do you select an element from the DOM?

To select an element from the DOM in JavaScript, you can use these methods:

1. document.getElementById() – Selects an element by its id.

2. document.querySelector() – Selects the first element that matches a CSS selector.

3. document.querySelectorAll() – Selects all elements that match a CSS selector.

what is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where you add a single event listener to a parent element instead of adding multiple listeners to individual child elements. The event listener on the parent catches events from its children as they "bubble up" (event bubbling).

Why is it useful?

1. Efficiency: Instead of adding multiple event listeners to many child elements, you attach one listener to the parent, which handles events for all children.

2. Dynamic elements: It works even for elements that are added to the DOM later, as the parent is always listening.

How do you manupulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM, you can do the following:

1. Use setAttribute() to change or add an attribute.

2. Use getAttribute() to read an attribute.

3. Use removeAttribute() to remove an attribute.
